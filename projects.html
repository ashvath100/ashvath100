Legged Robot Support For Ardupilot

The idea behind this project was to make a opensource platform for walking robots,where
anyone with an ardupilot flight contoller and basic know how can build a complex system
like a quadruped.
The support was implemented via the new scripting engine in ardupiot.
With the added support for PyBullet users can also modify a few parameters and add their custom frames simulate and test the physics.

Tnet

Tnet is an end to end solution with a portal designed to help customers view their injection moulding machine data in an intuitive web interface.
Uses MySQL, Node.js with Express and AngularJS. Part of my summer internship at Toshiba Machine, Chennai.
