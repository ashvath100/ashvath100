Legged Robot Support For Ardupilot

The idea behind this project was to make a opensource platform for walking robots,where
anyone with an ardupilot flight contoller and basic know how can build a complex system
like a quadruped.
The support was implemented via the new scripting engine in ardupiot.With the added
support for PyBullet users can also modify a few parameters and add their custom frames
simulate and test the physics.

Automated Guided Vehicle

During my time at Srinar Robotics at Chennai I worked on upgrading the existed frame work
which was being used to run the AGV by shifting the higher level control to ROS , spent my
time here experimenting on variaous mapping and SLAM methods.

Autonomous Underwater Vehicle (AUV)

One of the most challenging projects I have worked on ,the AUV is capable of completing tasks
autonomously with no theather or controller.The system was build over the ArduSub code base from
Ardupilot with an on board computer and cameras used for navigation.
